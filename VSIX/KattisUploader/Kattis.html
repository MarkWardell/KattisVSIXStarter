<link rel="stylesheet" href="/css/system.css"> <div class="problembody"> <p>The Intrusion and Crime Prevention Company (ICPC) builds intrusion detection systems for homes and businesses. The International Collegiate Programming Contest (in a strange coincidence also known as ICPC) is considering hiring the company to secure the room that contains the problem set for next year’s World Finals.</p> <p>The contest staff wants to prevent the intrusion attempts that were made in past years, such as rappelling down the outside of the building to enter through a window, crawling through air ducts, impersonating Bill Poucher, and the creative use of an attack submarine. For that reason, the problems will be stored in a room that has a single door and no other exits.</p> <p>ICPC (the company) proposes to install sensors on the four sides of the door, where pairs of sensors are connected by wires. If somebody opens the door, any connected sensor pair will detect this and cause an alarm to sound.</p> <p>The system has one design flaw, however. An intruder might cut the wires before opening the door. To assess the security of the system, you need to determine the minimum number of line segments that cut all wires. Figure&nbsp;1 shows two configurations of wires on the door (corresponding to the two sample inputs), and minimum-size cuts that intersect all wires.</p> <div id="fig:cut1" class="figure"> <center> <table cellspacing="0" class="tabular"> <tr> <td style="text-align:center"> <p><img src= "/problems/single/file/statement/en/img-0001.png" alt="\includegraphics[width=6cm]{sample1}" style= "width:6cm"></p> </td> <td style="text-align:center"> <p><img src= "/problems/single/file/statement/en/img-0002.png" alt="\includegraphics[width=6cm]{sample2}" style= "width:6cm"></p> </td> </tr> <tr> <td style="text-align:center"> <p>(a) Four wires (blue) that can be intersected with a single cut (red).</p> </td> <td style="text-align:center"> <p>(b) Five wires that need two cuts.</p> </td> </tr> </table> <div class="caption"> <b>Figure 1</b>: Illustrations of Sample Inputs 1 and 2. </div> </center> </div> <h2>Input</h2> <p>The input starts with a line containing three integers <span class="tex2jax_process">$n$</span>, <span class= "tex2jax_process">$w$</span>, and <span class= "tex2jax_process">$h$</span>, which represent the number of wires installed (<span class="tex2jax_process">$1 \leq n \leq 10^6$</span>) and the dimensions of the door (<span class= "tex2jax_process">$1 \leq w, h \leq 10^8$</span>). This is followed by <span class="tex2jax_process">$n$</span> lines, each describing a wire placement. Each of these lines contains four integers <span class="tex2jax_process">$x_1$</span>, <span class="tex2jax_process">$y_1$</span>, <span class= "tex2jax_process">$x_2$</span>, and <span class= "tex2jax_process">$y_2$</span> (<span class= "tex2jax_process">$0 \leq x_1, x_2 \leq w$</span>, <span class= "tex2jax_process">$0 \leq y_1, y_2 \leq h$</span>), meaning that a wire goes from <span class="tex2jax_process">$(x_1, y_1)$</span> to <span class="tex2jax_process">$(x_2, y_2)$</span>. Each wire connects different sides of the door. No wire is anchored to any of the four corners of the door. All locations in the input are distinct.</p> <h2>Output</h2> <p>Display a minimum-size set of straight line cuts that intersect all wires. First, display the number of cuts needed. Then display the cuts, one per line in the format <span class= "tex2jax_process">$x_1$</span> <span class= "tex2jax_process">$y_1$</span> <span class= "tex2jax_process">$x_2$</span> <span class= "tex2jax_process">$y_2$</span> for the cut between <span class= "tex2jax_process">$(x_1,y_1)$</span> and <span class= "tex2jax_process">$(x_2,y_2)$</span>. Each cut has to start and end on different sides of the door. Cuts cannot start or end closer than <span class="tex2jax_process">$10^{-6}$</span> to any wire anchor location or any corner of the door.</p> <p>Cuts may be displayed in any order. The start and end locations of each cut may be displayed in either order. If there are multiple sets of cuts with the same minimum size, display any of them.</p> <table class="sample" summary="sample data"> <tr> <th>Sample Input 1</th> <th>Sample Output 1</th> </tr> <tr> <td> <pre>4 4 6 0 1 4 4 0 5 2 0 0 3 3 6 2 6 4 2 </pre> </td> <td> <pre>1 0 4 4 3 </pre> </td> </tr> </table> <table class="sample" summary="sample data"> <tr> <th>Sample Input 2</th> <th>Sample Output 2</th> </tr> <tr> <td> <pre>5 4 6 0 2 2 0 0 3 2 6 1 6 3 0 1 0 4 4 3 6 4 2 </pre> </td> <td> <pre>2 0 4 4 4.5 0 1 4 1 </pre> </td> </tr> </table> </div>
